Description

The majority of the world speaks minority languages, and most of these minority languages don't have any speech technology. The Long Tail Language Competition aims to change this, allowing you to train models for this long tail.

OVH is generously providing compute resources to The Long Tail Language Competition, giving you free GPU time to train a speech recognition model for one (or more) of these long tail languages, while Coqui is providing code and data for you to train up such a model.

Registering for GPU Time

To obtain OVH's generously donated free GPU time for the competition, you should navigate to OVH's Discord registration page, fill in the required info, and join the coqui-ai-stt-challenge channel to obtain all the details.

Entering a Competition

To enter a competition, navigate to the Competitions tab on this page, search for the competition that interests you, then navigate to the competition's home page and register there.

The details of judging and how one competes in the competition are described on the competition's home page.
